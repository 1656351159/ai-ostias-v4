## 1. 这是什么

AI-OSTIAS = AI 驱动的科技情报采集系统。用户用**自然语言**下爬取任务 → **OpenClaw Agent** 调起爬虫 Skill 执行 → 过程与结果在 **Web 界面实时可见、可干预**。

版本演进：
- **V1** 自研模拟 Runtime（已废弃）
- **V2** crawl4more：单机 Python 时间片轮换多任务爬虫调度器（crawl4ai + LangChain + SQLite），调度内核验证通过但无 UI
- **V3** 以 OpenClaw Runtime 为核心的接入验证（Adapter→Agent 链路、安全门控），但"不可看"（只有 CLI）
- **V4（当前）**：V2 整理为独立 Skill + OpenClaw 调用 + Runtime 执行 + 可视化前端。**M1–M4 已完成并联调通过**

## 2. 总体架构

```
前端（React，3 标签页：对话任务/结果展示/系统状态，5s 轮询）
  │ REST /api/*
▼
Orchestrator（FastAPI，8 接口，异步提交任务）
  │ RuntimeAdapter.submit(task) -> RuntimeResult（唯一业务接口）
▼
OpenClaw Runtime（Gateway RPC，本机 ws://127.0.0.1:18789，Agent=researcher-v4）
  │ Agent 读 SKILL.md → 受控 exec 执行 run.py（exec 白名单仅 /usr/bin/python3）
▼
crawl4more Skill（TimeSliceScheduler → TaskManager → CrawlerWorker(crawl4ai+LLM 抽取)）
  │ 写三张表
▼
SQLite（crawl_jobs / crawl_subtasks / crawl_task_logs = 状态事实源，前端轮询它实现"可看"）
```

**人工干预原理**：前端改库（Job 状态/参数）→ Worker 每轮循环重读 DB → 生效。暂停靠 `status=paused + locked_by='manual_pause'` 加锁（防止调度器把 paused 当时间片挂起自动复活）。

## 4. 如何运行（本机）

```bash
# 1. 后端（终端 1）
cd orchestrator && .venv/bin/uvicorn main:app --port 8000
#    （.venv 不在交接包；重建：python3 -m venv .venv && .venv/bin/pip install -r requirements.txt）

# 2. 前端（终端 2）
cd frontend && npm install && npm run dev
#    浏览器打开终端提示的地址（默认 5173）

# 3. 使用：对话任务页填 URL + 自然语言描述（如"采集该站点科技情报，最多5页"）
#    → 解析参数 → 确认开始 → 队列监控实时刷新 → 结果展示页筛选/导出
```

mock 开关（界面默认开）= 演示模式，不碰真实网站、不需要爬虫依赖。关闭后才走真实爬取（需 §6 的依赖）。

## 5. 核心契约（改动任何东西前先读这里）

- **run.py 输出契约**：stdout **最后一行**必须是单行 JSON `{"job_id","status","processed_pages","extracted_count","failed_subtasks","error"}`，status ∈ `completed|failed|paused|cancelled`；日志一律走 stderr
- **API**：`POST /api/tasks/parse`、`POST /api/jobs`、`GET /api/jobs`、`GET /api/jobs/{id}/status`、`POST /api/jobs/{id}/control`（pause/resume/cancel/update）、`GET /api/results`（筛选/排序/分页）、`GET /api/results/export?format=csv|json`、`GET /api/system/status`
- **DB 三表**：crawl_jobs（作业）、crawl_subtasks（URL 单元）、crawl_task_logs（审计流水）；schema 在 `crawl4more-skill/db/schema.sql`
- **安全哲学**（不可违反）：工具硬策略在 OpenClaw Agent 侧强制（tools.allow=["read","exec"]，exec 白名单仅 /usr/bin/python3），Prompt 只是 advisory；任务不能动态声明新工具；所有对外输出递归脱敏
